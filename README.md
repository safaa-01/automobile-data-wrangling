# 🚗 Automobile Data Wrangling Project

## 📌 Overview
Complete data wrangling pipeline on the IBM Automobile dataset using Python.

## 🛠️ Steps Covered
- ✅ Missing value handling (mean imputation)
- ✅ Data type correction
- ✅ Standardization (MPG → L/100km)
- ✅ Normalization (Min-Max 0→1)
- ✅ Outlier detection (IQR)
- ✅ Feature engineering (power-to-weight ratio)
- ✅ Binning (horsepower & price)
- ✅ Encoding (ordinal + dummy variables)
- ✅ Correlation analysis
- ✅ Visualization (matplotlib & seaborn)
- ✅ Linear Regression to predict price

## 📊 Key Findings
- Engine size has the highest correlation with price (0.87)
- Horsepower strongly predicts price (0.81)
- Fuel consumption (L/100km) positively correlates with price

## 🧰 Technologies
Python | Pandas | NumPy | Matplotlib | Seaborn | Scikit-learn

## 📁 Dataset
[IBM Automobile Dataset](https://archive.ics.uci.edu/ml/datasets/automobile)
